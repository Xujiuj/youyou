# 🌹 Rose Page - 优优的秘密花园

## 设计理念

这是一个充满极致浪漫和诗意的过渡页面，作为密码验证后的中转站。

## 交互流程

1. **输入阶段** - 用户输入密码
   - 深邃星空背景，带有闪烁的星星
   - 浮动的心形装饰元素
   - 玻璃拟态风格的输入框
   - 渐变发光的提交按钮

2. **粒子爆炸过渡** - 密码正确后
   - 输入框化作200+个光粒子向四周飘散
   - 包含心形轨迹的特殊粒子
   - 粒子带有旋转、重力、阻力等物理效果
   - 多彩渐变色（粉色系为主）
   - 光晕和阴影效果

3. **选择阶段** - 显示两个精美按钮
   - **想对你说的话** - 跳转到主故事页面（可点击）
   - **优优的元宇宙** - 暂未开放，显示"施工中"标识

## 视觉特效

- 深邃的紫粉色渐变背景
- 闪烁的星空效果
- 浮动的心形装饰
- 玻璃拟态（Glassmorphism）设计
- 发光和阴影效果
- 流畅的动画过渡
- 悬停时的交互反馈

## 技术实现

- Canvas 2D 粒子系统
- GSAP 动画库
- CSS 渐变和滤镜
- 响应式设计
- TypeScript + React
